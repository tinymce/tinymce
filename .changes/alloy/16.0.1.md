## 16.0.1 - 2026-05-18

### Added
- New `onTab` and `onShiftTab` optional handlers on `TieredMenuSpec`. #TINY-13341
- Added new dragging mode based on pointer events. #TINY-14241

### Changed
- Tab behaviour inside an open dropdown menu now closes the menu instead of cycling through menu items. #TINY-13341

### Removed
- Removed the `moveOnTab` option from `MenuSpec`. #TINY-13341
