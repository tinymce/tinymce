キーボード ナビゲーションの開始

メニュー バーをフォーカス
Windows または Linux: Alt+F9
macOS: ⌥F9
ツール バーをフォーカス
Windows または Linux: Alt+F10
macOS: ⌥F10
フッターをフォーカス
Windows または Linux: Alt+F11
macOS: ⌥F11
コンテキスト ツール バーをフォーカス
Windows、Linux または macOS: Ctrl+F9

ナビゲーションは最初の UI 項目から開始され、強調表示されるか、フッターの要素パスにある最初の項目の場合は 下線が引かれます。

UI セクション間の移動

次の UI セクションに移動するには、Tab を押します。

前の UI セクションに移動するには、Shift+Tab を押します。

これらの UI セクションの Tab の順序:

  1. メニュー バー
  2. 各ツール バー グループ
  3. サイド バー
  4. フッターの要素パス
  5. フッターの単語数切り替えボタン
  6. フッターのブランド リンク
  7. フッターのエディター サイズ変更ハンドル

UI セクションが存在しない場合は、スキップされます。

フッターにキーボード ナビゲーション フォーカスがあり、表示可能なサイド バーがない場合、Shift+Tab を押すと、 フォーカスが最後ではなく最初のツール バー グループに移動します。

UI セクション内の移動

次の UI 要素に移動するには、適切な矢印キーを押します。

左矢印と右矢印のキー

  • メニュー バーのメニュー間で移動します。
  • メニュー内のサブメニューを開きます。
  • ツール バー グループのボタン間で移動します。
  • フッターの要素パスの項目間で移動します。

下矢印と上矢印のキー

  • メニュー内のメニュー項目間で移動します。
  • ツール バー ポップアップ メニュー内のメニュー項目間で移動します。

矢印キーで、フォーカスされた UI セクション内で循環します。

開いたメニュー、開いたサブメニュー、開いたポップアップ メニューを閉じるには、Esc キーを押します。

現在のフォーカスが特定の UI セクションの「一番上」にある場合、Esc キーを押すと キーボード ナビゲーションも完全に閉じられます。

メニュー項目またはツール バー ボタンの実行

目的のメニュー項目やツール バー ボタンが強調表示されている場合、リターン、Enter、 またはスペース キーを押して項目を実行します。

タブのないダイアログの移動

タブのないダイアログでは、ダイアログが開くと最初の対話型コンポーネントがフォーカスされます。

Tab または Shift+Tab を押して、対話型ダイアログ コンポーネント間で移動します。

タブ付きダイアログの移動

タブ付きダイアログでは、ダイアログが開くとタブ メニューの最初のボタンがフォーカスされます。

Tab または Shift+Tab を押して、このダイアログ タブの対話型コンポーネント間で移動します。

タブ メニューをフォーカスしてから適切な矢印キーを押して表示可能なタブを循環して、 別のダイアログに切り替えます。
