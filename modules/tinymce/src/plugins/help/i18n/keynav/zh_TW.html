開始鍵盤瀏覽

跳至功能表列
Windows 或 Linux：Alt+F9
macOS：⌥F9
跳至工具列
Windows 或 Linux：Alt+F10
macOS：⌥F10
跳至頁尾
Windows 或 Linux：Alt+F11
macOS：⌥F11
跳至關聯式工具列
Windows、Linux 或 macOS：Ctrl+F9

瀏覽會從第一個 UI 項目開始，該項目會反白顯示，但如果是「頁尾」元素路徑的第一項， 則加底線。

在 UI 區段之間瀏覽

從 UI 區段移至下一個，請按 Tab。

從 UI 區段移回上一個，請按 Shift+Tab。

這些 UI 區段的 Tab 順序如下：

  1. 功能表列
  2. 各個工具列群組
  3. 側邊欄
  4. 頁尾中的元素路徑
  5. 頁尾中字數切換按鈕
  6. 頁尾中的品牌連結
  7. 頁尾中編輯器調整大小控點

如果 UI 區段未顯示，表示已略過該區段。

如果鍵盤瀏覽跳至頁尾，但沒有顯示側邊欄，則按下 Shift+Tab 會跳至第一個工具列群組，而不是最後一個。

在 UI 區段之內瀏覽

在兩個 UI 元素之間移動，請按適當的方向鍵。

向左和向右方向鍵

  • 在功能表列中的功能表之間移動。
  • 開啟功能表中的子功能表。
  • 在工具列群組中的按鈕之間移動。
  • 在頁尾的元素路徑中項目之間移動。

向下和向上方向鍵

  • 在功能表中的功能表項目之間移動。
  • 在工具列快顯功能表中的項目之間移動。

方向鍵會在所跳至 UI 區段之內循環。

若要關閉已開啟的功能表、已開啟的子功能表，或已開啟的快顯功能表，請按 Esc 鍵。

如果目前已跳至特定 UI 區段的「頂端」，則按 Esc 鍵也會結束 整個鍵盤瀏覽。

執行功能表列項目或工具列按鈕

當想要的功能表項目或工具列按鈕已反白顯示時，按 Return、Enter、 或空白鍵即可執行該項目。

瀏覽非索引標籤式對話方塊

在非索引標籤式對話方塊中，開啟對話方塊時會跳至第一個互動元件。

按 Tab 或 Shift+Tab 即可在互動式對話方塊元件之間瀏覽。

瀏覽索引標籤式對話方塊

在索引標籤式對話方塊中，開啟對話方塊時會跳至索引標籤式功能表中的第一個按鈕。

若要在此對話方塊的互動式元件之間瀏覽，請按 Tab 或 Shift+Tab。

先跳至索引標籤式功能表，然後按適當的方向鍵，即可切換至另一個對話方塊索引標籤， 以循環瀏覽可用的索引標籤。
