Plugin: style

Adds CSS style editing support to TinyMCE, this will enable you to edit allmost any CSS style property in a visual way.

Installation Instructions

  • Add plugin to TinyMCE plugin option list example: plugins : "style".
  • Add the styleprops butto names to button list, example: theme_advanced_buttons3_add : "styleprops".

Initialization Example

tinyMCE.init({
	theme : "advanced",
	mode : "textareas",
	plugins : "style",
	theme_advanced_buttons3_add : "styleprops"
});
